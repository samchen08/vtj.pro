## ❓ 常见问题解答

### 版本区别与功能对比

#### Q1: 在线版源码与 Gitee 开源版本有什么区别？

> **开源版（Gitee）**：
>
> - 单应用开发工具，设计器嵌入前端开发环境运行
> - 数据采用本地文件存储
> - 设计器不支持在线部署
> - 适合个人学习和小型项目开发

> **在线版（高级服务）**：
>
> - 基于开源工具衍生的完整在线开发平台
> - 采用数据库存储，支持多应用管理
> - 支持在线部署、在线渲染和项目出码
> - 包含完整的用户权限和项目管理功能
> - 适合团队协作和企业级应用开发

#### Q2: 公有云服务版与私有化部署版有哪些区别？

> **公有云服务版**：
>
> - 完整的前端项目工程，安装依赖后即可运行
> - 后端对接 `app.vtj.pro` 公有云服务接口
> - 前端可独立部署到 Nginx 等静态服务器
> - 不包含管理后台模块
> - 适合快速启动和中小型项目

> **私有化部署版**：
>
> - 全功能前后端一体化项目工程
> - 后端采用 **NestJs** 框架，数据库为 **MySQL**
> - 包含完整的管理后台系统
> - 功能与 `app.vtj.pro` 平台完全一致
> - 支持完全自主可控的私有化部署
> - 适合对数据安全有要求的中大型企业

#### Q3: 源码使用是否有时间限制？

> - 源码使用**无时间限制**，一次购买永久使用
> - 在订阅期内提供**免费升级更新**服务
> - 订阅期满后仍可继续使用已获得的版本
> - 后续升级可选择续订或按需购买新版本

#### Q4: 源码能否用于商业项目？

> - **允许用于商业项目**，包括企业内部系统和客户项目
> - 支持二次开发和定制化修改
> - 无需支付额外的授权费用
> - 遵循项目开源协议的相关条款

### 技术架构与扩展能力

VTJ.PRO 采用模块化架构设计，核心功能包括：

1. **低代码设计器引擎**：基于 Vue3 + TypeScript 构建，支持可视化拖拽和源码级编辑
2. **双向代码转换**：独创的 DSL 与 Vue 源码双向编译技术，实现低代码与手写代码的自由切换
3. **AI智能辅助**：集成多种 AI 能力，支持自然语言生成组件、设计稿识别、代码优化等
4. **多平台渲染器**：支持 Web、H5、UniApp 等多端渲染
5. **可扩展物料体系**：提供丰富的组件库和模板，支持自定义组件开发和集成

### 部署与运维支持

#### 系统要求

- **Node.js**: v20.0.0 或更高版本
- **数据库**: MySQL 5.7+
- **操作系统**: Linux / Windows / macOS
- **内存**: 建议 2GB 或以上
- **存储**: 建议 50GB 或以上可用空间
