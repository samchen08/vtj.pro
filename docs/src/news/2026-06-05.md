# 开源 AI 低代码平台 VTJ.PRO 双版本齐发：核心引擎 v0.17.1 与在线平台 v2.4.1 正式上线，强化团队协作与 AI 资产管理

**广州 · 2026年6月5日** —— 面向专业开发者的开源 AI 驱动 Vue 低代码平台 VTJ.PRO 今日宣布，正式发布**核心引擎 v0.17.1**与**在线应用开发平台 v2.4.1** 两大版本更新。本次更新全面聚焦团队协作效率与 AI 开发资产的智能化管理，为开发者带来更流畅的多人协同体验和更灵活的数据处理能力，进一步夯实 VTJ.PRO 作为“AI 驱动的 Vue3 低代码平台”的生产级可靠性。

自 2.0 版本以来，VTJ.PRO 凭借“双向代码自由转换”机制与 AI 智能体（Agent + Skills）架构，在开源社区引发强烈反响，目前已在 Gitee 收获 10K+ Star，并荣获 Gitee 2025 年度“大前端 Top3”。VTJ.PRO 始终秉持“代码可编辑、逻辑可洞察、平台可脱离”的核心设计哲学，v2.4.1 与 v0.17.1 的更新再次印证了这一理念——通过赋能团队协作与资产沉淀，让企业级应用的开发流程更加高效可控。

### 👥 团队协作进阶：多人同步与资产复用双线升级

在线应用开发平台 v2.4.1 在团队协作层面实现了质的飞跃。**多人协助自动同步项目信息**功能正式上线，当多个团队成员同时在线开发同一应用时，系统会自动实时同步项目结构与数据变更，彻底消除了因信息滞后导致的手动刷新与沟通成本。团队可以像使用云端协作文档一样，在同一应用空间中并行开发、实时感知彼此的操作动态，让多人低代码协同开发真正变得流畅无感。

与协作能力相匹配的是资产管理能力的全面提升。**应用克隆功能**让开发者可以一键复制现有应用的完整配置与结构，用于快速搭建新项目或进行功能迭代测试。此外，平台的**共享模版生态**也进一步丰富——v2.3.6 版本已开放共享模版功能，开发者可将页面、模块或完整应用打包为模版发布到共享空间供团队复用，实现“一次发布，处处引用”。协作与复用双线打通，VTJ.PRO 正在构建一套完整的企业级团队开发闭环。

### 🎯 AI 资产精细化管理：多文件处理与对话记录导出

作为以 AI 能力为核心竞争力的低代码平台，VTJ.PRO 在本次更新中对 AI 相关的数据处理能力进行了系统性的精细化升级。

**AI 文件识别支持附加文本提示词**，开发者在通过 AI 上传设计稿截图、JSON 元数据等文件时，可以附加上下文说明，帮助 AI 更精准地理解需求意图，显著提升代码生成的准确率。同时，**AI 图片与 JSON 识别支持多文件批量上传**，一次性上传多张图片或多份元数据文件，AI 将逐个识别并整合生成完整页面代码，大大提升了批量处理的效率。

值得关注的是，**AI 对话记录现已支持导出为 Markdown 格式**。AI 助手是 VTJ.PRO 平台的核心交互入口之一，开发者可以通过自然语言对话完成代码生成、组件修改、设计稿转代码等操作。v2.4.1 上线后，开发者可以将完整的 AI 对话历史导出为标准 Markdown 文件，便于技术复盘、需求文档沉淀以及跨团队的知识共享，让每一次 AI 协作都成为可追溯、可复用的资产。

### 🛠️ 核心引擎 v0.17.1：为 AI 与协作能力提供底层支撑

同步发布的**核心引擎 v0.17.1** 为平台的最新能力提供了稳定的底层技术支撑。VTJ.PRO 的低代码引擎利用领域特定语言（DSL）模式来抽象应用逻辑、UI 结构和状态管理，通过先进的代码生成和解析能力，弥合了可视化工作台与标准 Vue 开发之间的鸿沟。v0.17.1 版本进一步优化了 DSL 与 Vue 源码之间的双向转换效率，并提升了多人协作场景下的数据同步性能，确保 AI 生成的代码变更能够精准、及时地响应到每个团队成员的设计器中。

### 📦 技术栈与体验方式

VTJ.PRO 基于 Vue 3 + TypeScript + Vite 构建，深度集成 ElementPlus、ECharts 等主流开源生态，并已接入 DeepSeek、Qwen、Gemini、GPT 等 10+ 款大模型。平台支持 Web、H5、UniApp 多端输出，同时提供完整的私有化部署方案，支持源码交付，充分满足企业级数据安全与自主可控的要求。
