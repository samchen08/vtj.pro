## `03` 三步上手，快速入门VTJ开发

**环境准备**

VTJ要求Node版本必须是v20+，建议使用nvm切换Node版本。这是唯一的硬性要求，非常简单。

**快速创建项目**

VTJ提供了项目脚手架，可快速创建新项目。只需执行一条命令：

```bash
npm create vtj@latest -- -t web
```

根据你的目标平台，可以选择创建Web应用（PC端）、H5应用（移动端）或uni-app（跨端应用）

让我们来看一个实际案例——构建一个用户管理系统后台界面。
