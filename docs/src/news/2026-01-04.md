## 新平台上线

VTJ.PRO 应用开发平台 V2.0 已正式上线，架构全面升级。

:::warning 重要更新

- 新平台启用域名 https://app.vtj.pro
- 原域名 app.vtj.pro 已重定向到新平台。旧平台将在 2026-03-31 停止服务。
  ::::

### 迁移指引

**本地版：**

重新安装依赖更新到最新版本，使用命令：

```bash
npm run clean && npm install
```

**在线版：**

1. 登录新平台( https://app.vtj.pro/#/workbench/service ) 下载公有云服务版本。
2. 如果继续使用原来的版本，需要在 `env.json` 文件 配置 `REMOTE` 和 `AUTH_CODE`, 如：

   ```json
   {
     "AUTH_CODE": "在新平台获取登录签名",
     "REMOTE": "https://app.vtj.pro",
     "BASE_PATH": "/",
     "OUT_DIR": "dist"
   }
   ```

   该方式仅允许在 2026-03-31 前使用。需尽快迁移到新平台。

3. 如已实现了后端服务完全私有化，不受本次更新影响。
