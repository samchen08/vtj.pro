# VTJ.PRO 应用开发平台 V2.0 上线公告

经过三个多月的紧张开发和全面测试，我们很高兴地宣布 **VTJ.PRO 应用开发平台 V2.0** 正式上线！本次版本升级是对平台前后端架构的**完全重构**，旨在为用户带来更强大、更稳定、更智能的开发体验。

## 🚀 V2.0 核心升级亮点

V2.0 版本基于最新的 **Vue3 + TypeScript + Vite** 技术栈进行全面重构，主要改进包括：

### 1. **架构全面升级**

- **前后端完全重构**：采用现代化架构设计，提升系统性能和可维护性
- **模块化设计**：增强代码复用性和扩展性，支持更灵活的定制开发
- **性能优化**：页面加载速度提升 40%，设计器响应时间缩短 60%

### 2. **AI 能力显著增强**

- **多模型支持**：Gemini X DeepSeek 双模型驱动
- **智能代码修复**：AI-CodeFix 引擎可自动检测并修复数十类代码问题
- **设计稿识别精度提升**：Figma/Sketch 设计稿转代码准确率提升至 95%+

### 3. **企业级功能完善**

- **国际化可视化管理**：支持 vue-i18n 的多语言词条可视化配置
- **精细化权限控制**：更灵活的 RBAC 权限管理体系
- **环境变量配置**：支持开发、测试、生产多环境部署

### 4. **开发体验优化**

- **双向代码转换优化**：导出代码质量更高，更符合团队编码规范
- **协作功能增强**：支持团队项目管理和版本控制

## 🔄 平台迁移与调整

为确保用户平稳过渡到新平台，我们将进行以下调整：

### 域名与访问方式变更

1. **新平台地址**：`https://app.vtj.pro`（主域名）
2. **旧平台重定向**：`app.vtj.pro` 将自动重定向至新平台
3. **访问方式调整**：
   - 旧平台 `app.vtj.pro` 自即日起**禁止账号密码登录**
   - 仅支持通过 `AUTH_CODE` 进行静默登录
   - **旧平台将于 2026年3月31日 正式停止服务**

### 本地版本更新

使用本地版 VTJ.PRO 的用户需要：

使用以下命令，重新安装项目依赖，更新到最新版本。

```bash
npm run clean && npm install
```
